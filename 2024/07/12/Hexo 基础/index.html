软件准备

在安装 Hexo 框架之前，需要先安装 Git 和 Node.js，因为它们是 Hexo 正常运行所必须的依赖软件。

  • Git：用于版本控制和部署静态网站。Git 官网下载
  • Node.js：提供运行 Hexo 所需的 JavaScript 环境和包管理工具。 Node.js 官网下载

安装完 Git 和 Node.js 后，就可以使用 npm 命令来安装 Hexo 及其相关插件，从而搭建和管理博客。

Hexo 的安装与初始化

  • 安装：在终端执行命令：npm install -g hexo-cli
  • 初始化：新建一个空文件夹，在该目录下执行命令：hexo init

新建的这个空文件夹就是博客的根目录。完成后，执行命令 hexo s 检查服务是否启动成功，在浏览器即可打开页面： http://localhost:4000/

新建完成后，指定文件夹目录下有：

  • node_modules: 依赖包
  • public：存放生成的页面
  • scaffolds：生成文章的一些模板
  • source：用来存放你的文章
  • themes：主题
  • _config.yml: 博客的配置文件

通过修改里面的文件就可以管理和配置博客了。

使用快捷键：Ctrl+C 即可停止服务。

Github 部署

创建一个新的 GitHub 仓库
