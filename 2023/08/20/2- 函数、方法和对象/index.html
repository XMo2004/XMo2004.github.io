JavaScript 函数、方法和对象

函数

函数使你可以将一系列语句组织成一个整体，以执行某一特定任务。如果在不同的地方有些任务要重复执行，可以重用函数（而非重复一组相同的语句）。

以上案例中，用户在页面顶端显示一条消息。此消息由一个 id 属性为 message 的 HTML 元素保存，此消息将被 JavaScript 修改。

函数的声明

声明函数的一般方法

要创建函数，先对函数命名，然后在花括号里编写完成任务所需的语句，这就是函数的声明。

  • 用 function 关键字声明函数。
  • 给函数指定名称（有时也称为标识符），接着是一对括号。
  • 花括号内是执行任务的语句。

声明需要信息的函数

有时，函数需要特定的信息来执行任务。在这种情况下，声明函数时要给它提供形参。在函数内部，形参的行为类似于变量。

  • 如果函数需要一些信息，在函数后面的括号内指定。
  • 括号中的这些项即函数的形参。在函数体中它们的行为和变量一样。

函数的调用

声明了函数后，就可以用一行代码执行花括号里的所有语句，这就是函数的调用。

  • 在函数名后面带上一对括号。这就是编程术语中的调用函数。
  • 可以在同一个 JavaScript 文件中调用同一个函数任意次。

调用需要提供信息的函数

调用带有形参的函数时，需在函数名后面的括号中指定一些值。这些值就是实参，可以像变量一样被赋值。
